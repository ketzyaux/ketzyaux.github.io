# Portafolio Web — Ketzya UX

Portafolio personal de UX Design construido con HTML, CSS y JavaScript puro.

## Tecnologías
- HTML5 semántico
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript Vanilla (ES6+)
- Google Fonts (Playfair Display + Inter)
- Tabler Icons

## Deploy
Sitio estático desplegado en GitHub Pages.

## Estructura
```
├── index.html
├── css/styles.css
├── js/main.js
├── assets/
│   ├── images/hero-portrait.jpg
│   └── docs/ketzya-cv.pdf
└── .nojekyll
```

## Desarrollo local
```bash
npx serve .
```
